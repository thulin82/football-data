# [Football Data API](https://github.com/thulin82/football-data-api)

Setup
---------------------
### Get project
```
$ git clone git@github.com:thulin82/football-data-api.git
```
### Get dependencies
```
$ npm install
```
### Run app
```
$ npm start
```

### Environment variables
In config/config.env add
```
API_KEY = 
```
